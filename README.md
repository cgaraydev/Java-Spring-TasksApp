features:

* Java
* Spring Boot
* Spring MVC
* Spring Data JPA
* Spring Security
* Hibernate
* Thymeleaf
* HTML
* CSS (Bootstrap)
* JavaScript
* MySQL
* Maven

screens: Login, Dashboard, Task List, Task Details, Manage Tasks

Esta app consiste en un sistema de gestión de tareas desarrollado con Spring Boot y Thymeleaf. Permite crear, editar, eliminar y visualizar tareas, además de gestionar autenticación y autorización de usuarios mediante Spring Security.
